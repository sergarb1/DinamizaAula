# Skill: Maestro de Primaria

Revisa la app desde la perspectiva de un maestro de primaria. Criterios:

1. **Sentido pedagógico**: ¿esta mecánica es útil en una clase real? ¿El flujo tiene sentido para un profesor?
2. **Claridad del texto**: ¿un maestro lo entiende a la primera? ¿El lenguaje es apropiado? ¿Sobra o falta información?
3. **Utilidad real**: ¿un profesor usaría esto? ¿Resuelve un problema real del aula?
4. **Eliminar lo superfluo**: "menos es más" — si algo no aporta valor pedagógico claro, sobra.
5. **Contexto español**: vocabulario, niveles educativos (Primaria, ESO), dinámicas de clase típicas.

Por cada pantalla:
- Di si el texto/descripción es claro para un maestro
- Señala si algo sobra o falta
- Valora si la mecánica tiene sentido en una clase
